# OSM2terrn Code of Conduct

## Purpose

This project is open to contributors of all backgrounds and skill levels. The goal is to keep interactions friendly, respectful, and focused on improving the project.

We encourage open discussion, constructive feedback, and collaboration. Everyone is welcome as long as they contribute in good faith and with mutual respect.

## Guidelines

Positive behaviors include:
- Being helpful and respectful when discussing ideas or giving feedback.
- Keeping conversations technical and on-topic.
- Offering constructive suggestions when reviewing code or issues.
- Acknowledging different approaches and working towards consensus.

Behaviors to avoid:
- Personal attacks, insults, or aggressive arguments.
- Releasing private information of others without consent.
- Disruptive behavior that derails project discussions.

## Moderation

If a discussion becomes unproductive or disrespectful, community maintainers have the right to intervene. This may include locking threads, moderating comments, or in extreme cases, restricting contributors who repeatedly cause disruption.

## Scope

This Code of Conduct applies to all project-related spaces including GitHub discussions, issues, pull requests, and any other collaborative platforms used.

## Reporting

If you encounter behavior that goes against these guidelines, you can report it to the project maintainers by opening a private discussion or issue.

## Final Notes

We aim to keep OSM2terrn a collaborative, enjoyable space. Please contribute with the intention of helping improve the project and supporting fellow developers.
